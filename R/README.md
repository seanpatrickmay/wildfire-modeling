# R Scripts

These R scripts post-process Earth Engine exports into final GOFER products.

- 2 - eePro_ParamSens.R: analyze parameter sensitivity output.
- 4 - eePro_FireProg.R: assemble fire progression outputs.
- 6a - eePro_cFireLine.R: post-process concurrent active fire lines.
- 6b - eePro_rFireLine.R: post-process retrospective active fire lines.
- 8 - table_GOFERstats.R: build GOFER summary stats table.
- 9 - make_GOFERfinal.R: assemble final GOFER data products.
- globalParams.R: shared parameters for the R pipeline.

fireData.csv is the R-side fire metadata table.
